## PARTE 1: VISIÓN GENERAL DEL SISTEMA

### ¿Qué es CreatorVault en Palabras Simples?

CreatorVault es una plataforma que actúa como **intermediario entre creadores de contenido y marcas que quieren publicidad.**

**Analogía:** Es como un agente de viajes, pero para creadores. El agente no vende viajes, conecta a turistas con hoteles y obtiene comisión.

### Los Tres Actores Principales

1. **Creadores:** Personas con audiencia en TikTok, Instagram, YouTube
2. **Marcas:** Empresas que quieren publicidad (Nike, Skillshare, Audible)
3. **CreatorVault:** Nosotros (el intermediario que conecta ambos)

### El Flujo de Dinero Simple

```
MARCA (tiene dinero)
    ↓
Deposita dinero en CreatorVault
    ↓
CREATORVAULT (retiene comisión)
    ↓
Distribuye dinero a creadores
    ↓
CREADOR (recibe dinero)
```

### ¿Qué es un Contrato Digital?

Es un documento que especifica exactamente qué debe hacer el creador y qué va a recibir.

```
CONTRATO DIGITAL - NIKE ZAPATILLAS

Partes:
- Nike (Marca)
- Juan Pérez (Creador)
- CreatorVault (Intermediario)

Términos:
1. Nike pagará: $1,000

   - Duración: 15-60 segundos

5. Si Juan no cumple: Nike recibe reembolso

## PARTE 8: CÓMO FUNCIONA EL PAGO

### Paso 1: Sistema Calcula Comisión

**Lo que pasa detrás:**

```
Dinero recibido de Nike: $50,000
Creadores a pagar: 50
Dinero por creador: $1,000

Comisión de CreatorVault: 20% de $50,000 = $10,000
Dinero para creadores: $50,000 - $10,000 = $40,000
Dinero por creador: $40,000 ÷ 50 = $800

Desglose:
- Nike pagó: $1,000 por creador
- CreatorVault comisión: $200 por creador
- Creador recibe: $800 por creador
```

### Paso 2: Sistema Transfiere Dinero

**Lo que pasa detrás:**

El sistema usa Stripe (empresa de pagos) para transferir dinero:

```
Paso 1: Dinero está en escrow ($50,000)
Paso 2: Sistema verifica que creadores cumplieron
Paso 3: Sistema libera dinero del escrow
Paso 4: Sistema transfiere $800 a cada creador
        - Usa información bancaria guardada
        - Transfiere a cuenta del creador
        - Tarda 1-2 días hábiles
Paso 5: Sistema retiene $10,000 para CreatorVault
```

### Paso 3: Creador Recibe Dinero

**Lo que hace el creador:**
1. Recibe email: "Pago completado - $800"
2. Inicia sesión en CreatorVault
3. Va a "Payment History"
4. Ve:

```
Pago Completado
Tipo: Patrocinio Nike
Monto: $800
Fecha: 25 de Enero 2026
Estado: Completado
```

5. Abre su cuenta bancaria
6. Ve: "+$800" en su cuenta

## PARTE 9: PAGO AUTOMÁTICO MENSUAL

### ¿Qué es el Pago Automático?

Es un pago que CreatorVault hace automáticamente cada 1 de mes a todos los creadores.

### Cómo Funciona

**El 1 de Enero a las 00:00 UTC:**

```
Paso 1: Sistema despierta (cron job)
Paso 2: Sistema calcula ingresos de cada creador:
        - Juan Pérez (Tier 2): $1,000 garantizado
        - Patrocinios completados: $800 (Nike) + $900 (Skillshare) = $1,700
        - Total: $1,000 + $1,700 = $2,700

Paso 3: Sistema crea payment record:
        - Creador: Juan Pérez
        - Monto: $2,700
        - Tipo: Guaranteed Income + Sponsorships
        - Fecha: 1 de Enero 2026

Paso 4: Sistema transfiere dinero:
        - Usa información bancaria guardada
        - Transfiere $2,700 a cuenta de Juan
        - Tarda 1-2 días hábiles

Paso 5: Sistema envía notificación:
        - Email: "Pago de $2,700 procesado"
        - App notification: "¡Pago recibido!"
        - SMS (opcional): "Tu pago de $2,700 está en camino"

Paso 6: Sistema actualiza dashboard:
        - Juan ve: "Pago completado - $2,700"
        - Juan ve: "Próximo pago: 1 de Febrero"
```

### ¿Qué Pasa si el Creador No Tiene Patrocinios?

```
Juan no hizo patrocinios este mes.
Sistema calcula:
- Ingreso garantizado: $1,000
- Patrocinios: $0
- Total: $1,000

Sistema transfiere: $1,000 a Juan
Juan recibe: $1,000 (aunque no hizo nada)

Eso es el "ingreso garantizado"
```

## PARTE 11: CÓMO SE GARANTIZAN LOS PAGOS

### Problema: ¿Qué Pasa si Nike no Paga?

**Solución: Depósito en Escrow**

```
Escrow = Cuenta de terceros donde se congela el dinero

Paso 1: Nike quiere hacer campaña de $50,000
Paso 2: Nike deposita $50,000 en escrow (no en CreatorVault)
Paso 3: El dinero está congelado, nadie puede tocarlo
Paso 4: Creadores hacen contenido
